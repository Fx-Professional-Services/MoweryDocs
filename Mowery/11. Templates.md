## Templates

Use **Templates** to manage preconfigured equipment and material lists for estimates and proposals. Templates help standardize option content and speed up quote creation.

---

## Open Templates

1. From the **Main Menu**, click **Templates**.
    
2. The **Templates** screen appears.

## Templates Screen Overview

### Left Panel — Line Items

The left side shows the template’s itemized components.

**Columns include:**

- **QTY** — Quantity of the item
    
- **Part Numbers** — Internal or vendor part number
    
- **Description** — Item name or details
    
- **Cost** — Internal cost reference
    
- **Sort** — Controls item order within the template
    
- **Delete icon** — Removes the line item
    

Use this list to review and organize all materials included in the template.

---

### Right Panel — Specification’s Estimates

This section contains the customer-facing description that appears in proposals.

It typically includes:

- System summary
    
- Equipment details
    
- Included components
    
- Warranty information
    
- Scope notes (tax, labor, permits, etc.)
    

**Best practice:** Keep this text clear and customer-friendly since it appears on printed reports.

---

## Navigate Templates

Use the buttons at the top of the screen:

- **Previous / Next** — Move between template records
    
- **Show All** — Display all templates
    
- **Create New** — Create a new template
    
- **Reserialize** — Reorder sort values (admin use)
    

---

## Create a New Template

1. Click **Create New**.
    
2. Enter the template information.
    
3. Add line items as needed.
    
4. Enter or edit the **Specification’s Estimates** text.
    
5. Save the record.
    

---

## Edit Line Items

To modify items in a template:

1. Locate the item in the left panel.
    
2. Update **QTY**, **Part Number**, **Description**, or **Cost** as needed.
    
3. Adjust the **Sort** value to change display order.
    

**Tip:** Lower sort numbers appear first.

---

## Edit Sort Numbers to Reorder Items

You can manually rearrange items in a template by editing their **Sort Number** field. This allows precise placement of new items without affecting existing ones.

### Steps

1. From the **Main Menu**, click **Templates**.
    
2. Locate the **Sort Number** column next to each item.
    
3. Click into a Sort Number field and enter a new number:
    
    - To insert an item between `6` and `7`, enter `6.1`.
        
    - You can use additional decimal places as needed (for example, `6.11`, `6.12`).
        

> The list updates immediately based on the new sort values. Descriptions and item details remain unchanged.

---

## Reserialize the List

Use the **Reserialize** button to simplify numbering after editing. This replaces any decimal values with a clean, sequential list.

### Steps

1. Click **Reserialize** at the top of the template list.
    
2. The system renumbers the entire list using whole numbers:
    
    `1 → 2 → 3 → 4 → ...`
    
3. Items retain their current positions; only the numbering changes.
    

> **Note:** The item order is preserved—only the numbering format is updated.

---

## Applies to All Templates

- Decimal-based sort numbers work across **all templates** in the system.
    
- There is **no limit** to the number of decimal places you can use.
    
- You can switch between decimal and integer-based order at any time using the **Reserialize** button.
    

---

## Delete a Line Item

1. Find the item in the list.
    
2. Click the **red delete icon** on that row.
    
3. Confirm the deletion if prompted.
