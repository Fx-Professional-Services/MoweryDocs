# Vendors User Guide

## Overview

Use the **Vendors** screen to:

- Create a new vendor
    
- View or edit vendor details
    
- Search for vendors
    
- Delete vendors
    
- Add notes and account information

---

# Navigate the Vendors screen

1. From the main menu, click **Vendors**.

# Create a new vendor

1. Click **New Vendor**.
    
2. In the **Business** section, enter:
    
    - **Business Name**
        
    - **Title** (if applicable)
        
    - **Street**
        
    - **City / State / Zip**
        
    - **Phone Type/Number**
        
    - **Fax Number**
        
    - **Email**
        
3. (Optional) Complete the **Personal** section if the vendor is an individual:
    
    - Title
        
    - First Name
        
    - Middle Name
        
    - Last Name
        
    - Street
        
    - City / State / Zip
        
    - Country
        
4. In the **Notes** panel, enter:
    
    - Account number
        
    - Special instructions
        
    - Internal comments
    

---

# Edit a vendor

1. Use one of the following methods to locate the vendor:
    
    - Click **Find**
        
    - Use **QuickFind Search**
        
    - Navigate using the record arrows

	* Click **List View**
	
2. Open the Vendor's Record by clicking the left arrow. 

3. Update the required fields.
    

The **Modified By** and timestamp fields update automatically.

---

# Search for a vendor

## Quick search

1. Enter a keyword in **QuickFind Search**.
    
2. Press Enter.
    

Use this method for fast lookups by:

- Business name
    
- Phone number
    
- Account number
    

## Advanced search

1. Click **Find**.
    
2. Enter search criteria.
    
3. Run the search.
    

To view all records, click **Find All**.

---

# Delete a vendor

1. Locate the vendor.
    
2. Click the X icon.
    
3. Confirm the deletion when prompted.

---

# Use record navigation

Use the navigation arrows in the top-right corner to:

- Go to the first record
    
- Go to the previous record
    
- Go to the next record
    
- Go to the last record
    

The record counter displays:

Record X of Y (Total)

---

# Field reference

## Business section

|Field|Description|
|---|---|
|Business Name|Legal or registered vendor name|
|Title|Contact title|
|Street|Business address|
|City/State/Zip|Location details|
|Phone Type/Number|Work, Fax, Pager|
|Fax Number|Vendor fax|
|Email|Vendor email address|

## Personal section

Use this section when the vendor is an individual instead of a company.

## Notes panel

Use Notes to store:

- Account numbers
    
- Special billing instructions
    
- Internal reference codes
    

Example:

ACCOUNT NO. GJR045

---

# Audit information

The bottom panel displays:

- **Created By**
    
- Creation date and time
    
- **Modified By**
    
- Last modified date and time
    

These fields are system-generated.
