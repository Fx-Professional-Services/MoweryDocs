# Vendors 

## Overview

The **Vendors screen** allows you to store and manage information about companies or individuals that supply products or services to the business.

Use this screen to:

- Create new vendor records
    
- Search for existing vendors
    
- Store contact information
    
- Track vendor account numbers
    
- Maintain notes about vendor relationships
    

This information is typically used when creating purchase orders or managing inventory suppliers.

---

# Screen Layout

The Vendors screen contains the following main sections:

1. Navigation and search tools
    
2. Business information
    
3. Vendor contact details
    
4. Personal contact information
    
5. Notes
    
6. Record history
    

Each section stores different types of vendor information.

---

# 1. Find or Navigate Vendor Records

Use the toolbar at the top of the screen to locate vendors.

### Buttons

|Button|Purpose|
|---|---|
|**List View**|Displays a list of all vendors|
|**New Vendor**|Creates a new vendor record|
|**Find**|Search for a specific vendor|
|**Find All**|Displays all vendor records|
|**Delete**|Removes the current vendor record|

### QuickFind Search

Use **QuickFind Search** to quickly locate a vendor.

Steps:

1. Click inside the **QuickFind Search** box.
    
2. Type part of the vendor name.
    
3. Press **Enter**.
    

The system displays the matching vendor record.

### Record Navigation

Use the arrow buttons in the top right to move between vendor records.

- First vendor
    
- Previous vendor
    
- Next vendor
    
- Last vendor
    

---

# 2. Add a New Vendor

To create a vendor record:

1. Click **New Vendor**.
    
2. Enter the vendor’s **Business Name**.
    
3. Enter the **street address** and **city/state/zip**.
    
4. Add contact information such as phone and email.
    
5. Enter any vendor notes.
    
6. Save the record.
    

Adding vendors ensures the company can easily reference suppliers when creating purchase orders.

---

# 3. Enter Business Information

The **Business** section stores company-level details.

|Field|Description|
|---|---|
|**Business Name**|Name of the vendor company|
|**Title**|Vendor contact title (if applicable)|
|**Street**|Business street address|
|**City / State / Zip**|Vendor location|

Use this section for vendor companies rather than individual contacts.

Example:  
A plumbing supply company providing installation parts.

---

# 4. Add Contact Information

This section stores vendor communication details.

|Field|Description|
|---|---|
|**Phone Type / Number**|Vendor phone contact|
|**Fax**|Vendor fax number|
|**Pager**|Alternate contact method|
|**Fax Number**|Additional fax information|
|**Email**|Vendor email address|

### Best practice

Enter at least **one reliable contact method** so staff can place orders or request support quickly.

---

# 5. Enter Personal Contact Information

The **Personal** section stores information for a specific vendor contact person.

Use this section if you work directly with an individual at the vendor company.

|Field|Description|
|---|---|
|**Title**|Contact title|
|**First Name**|Contact first name|
|**Middle Name**|Contact middle name|
|**Last Name**|Contact last name|
|**Street**|Personal address|
|**City / State / Zip**|Contact location|
|**Country**|Contact country|

Example uses:

- Sales representative
    
- Account manager
    
- Vendor support contact
    

---

# 6. Use the Notes Section

The **Notes** area stores additional vendor details.

Common information stored here includes:

- Vendor account numbers
    
- Payment terms
    
- Order instructions
    
- Contact preferences
    

Example note:

Account No: GJR045  
Preferred supplier for plumbing parts

This section helps staff remember important vendor details when placing orders.

---

# 7. Record History

At the bottom of the screen, the system tracks who created and modified the vendor record.

|Field|Description|
|---|---|
|**Created By**|User who created the record|
|**Date Created**|When the record was created|
|**Modified By**|User who last edited the record|
|**Date Modified**|When the record was last updated|

This information helps track changes to vendor data.

---

# Common Tasks

## Find a Vendor

1. Type the vendor name in **QuickFind Search**.
    
2. Press **Enter**.
    
3. Review the displayed record.
    

---

## Update Vendor Information

1. Locate the vendor using **QuickFind Search**.
    
2. Update the necessary fields.
    
3. Save the changes.
    

---

## Record Vendor Account Numbers

1. Open the vendor record.
    
2. Enter the account number in the **Notes** section.
    
3. Save the record.
    

This makes the account number available when placing orders.

---

# Best Practices

### Keep vendor information current

Update phone numbers and addresses whenever they change.

### Record account numbers

Store vendor account numbers in the **Notes** section for quick reference.

### Use consistent vendor names

Enter vendor names consistently so searches work reliably.
