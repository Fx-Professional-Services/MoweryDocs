# 🛠️ Service Calls

The **Service Calls** section is where you manage all service jobs. These jobs may include installation, repair, maintenance, or inspection for HVAC, plumbing, or other customer services.

</p>

### 🔍 Purpose

A **Service Call** is a scheduled visit to a customer’s property for a specific service task.

### 🛠️ Key Features

- Sort and manage scheduled jobs
    
- Create or edit service calls
    
- Print service history and tickets
    
- View active jobs at a glance
    
- Log service-related communications, including technician texts
    

With these tools, you can:

- Assign technicians from the Dispatch Board
    
- Send technician details to customers by text
    
- Provide customers with a secure Technician Profile Page
    
- Print job-specific parts lists
    
- Manage technician and staff records
    
- Track auto-logout behavior
    
- Organize categories and subcategories for products

## 🧑‍🔧 Assign Technicians Using the Dispatch Board

The **HVAC Schedule Board** lets dispatchers assign technicians to unassigned service calls quickly.

### Assign a Technician to a Service Call
To assign a technician from the **Schedule Board**:

1. From the main menu, select **Service Calls**.
    
2. The **Schedule Board** opens. Unassigned HVAC calls appear in the **Unassigned** column.
    
3. Select a call to open the **Service Call Detail** screen.
    
    - Or click **List View** (upper left).

  <img src="https://github.com/Fx-Professional-Services/MoweryDocs/blob/e222dbdf97bf98fc4f6b7d7e6988d30a856c3900/Mowery/Mowery%20User%20Guide/Images/list_view_assign_technician.png" 
       alt="List View" style="width: 75%;" />
</p>


- In the **Status** column, select **Scheduled Service Calls**, then click the arrow.
    
<p style="float: right; margin-left: 15px; width: 20%;">
  <img src="https://github.com/Fx-Professional-Services/MoweryDocs/blob/e222dbdf97bf98fc4f6b7d7e6988d30a856c3900/Mowery/Mowery%20User%20Guide/Images/arrow_list_view.png" 
       alt="Expand Arrow" style="width: 75%;" />
</p>

4. In **Step 2: Assign Technician**, click **New**.
    
5. In the **Technician** field, select a technician from the dropdown.
    
6. Click **Confirm** (blue arrow) to record the dispatch time.

- If two technicians are assigned, confirm both before sending a text.
        
7. Click **Dispatch** (green arrow) to update the status to _Dispatch_ and log the time.
    
8. Click **Send Text** (blue phone icon) to send job details and log the message time.

> **Note:** Only configured technicians appear in the dropdown list.

## 📱 Send a Text Message with Technician Details

You can send customers a text with technician and appointment details after certain requirements are met.

### 🔒 Requirements

- The service call must be **confirmed**.
    
- A technician must be **assigned**.
    
    - If no technician is assigned, clicking the icon shows:  
        _“No Technician selected. Please assign a technician before sending a message.”_
        
- The customer’s phone number must be:
    
    - Entered on the **Master Property List**
        
    - Marked as **Preferred**
        
    - Checked for **Accept Text Message**
        
    - A valid 10-digit U.S. number

### ✅ Send a Text Message

1. From the main menu, go to **Service Calls**.  
2. Open the scheduled and confirmed **service call**.  
3. In **Step 2: Assign Technician**:  
   - Verify a technician is assigned  
   - Confirm the call by entering the confirmation time via the blue arrow  
4. In the **Contact** section:  
   - Select **Text**
   - Verify the number is valid, marked **Preferred**, and checked for **Accept Text Message**
5. Click **Send Text Message** (blue phone icon).  
6. Wait for the confirmation or an error message.

### ✉️ Message Contents 

The customer receives a message with:

- **Technician** name  
- **Customer address**  
- Link to the **Technician Profile Page**  
- Technician introduction  
**Example:**  
Hello [Customer Name], [Technician Name] from Mowery is on the way to [Customer Address].
Click here for more details: [Technician Profile Webpage Link]

Hi, my name is [Technician Name]. I’ll be your technician today. [Technician Bio] If you have any questions, I’m here to help. Thank you for choosing Mowery!

### 🔤 Template Variables

| Variable                            | Description                   |
| ----------------------------------- | ----------------------------- |
| `[Customer Name]`                   | Full name of the customer     |
| `[Technician Name]`                 | Assigned technician’s name    |
| `[Customer Address]`                | Service location              |
| `[Technician Bio]`                  | Short technician introduction |
| `[Technician Profile Webpage Link]` | Secure profile link           |

### ⚠️ Error Messages 

| Condition                                 | Message                                                                |
| ----------------------------------------- | ---------------------------------------------------------------------- |
| No technician assigned                    | **No Technician selected. Please assign a technician before sending.** |
| No preferred number                       | A preferred phone number is required to send a message.                |
| Accept Text Message checked but no number | Cell number is required if Accept Text Message is checked.             |

## ✅ How a Customer Confirms the Service Call

When customers receive a service notification, it includes a link to the **Technician Profile Page**. Clicking the link confirms the appointment and logs the confirmation with a timestamp. Each link is unique and expires once the call is closed.
### 🔍 What Customers See

**Service Information**  
- Service status (Scheduled, In Progress, Completed)  
- Appointment time  

**Technician Details**  
- Technician name  
- Technician photo

## 📄 Send a Parts List from a Service Call

Dispatchers can email a parts request directly from the Service Call screen. This notifies the parts department quickly and records the request digitally.

### ✅ What Happens When You Click Send to Parts 

When a dispatcher clicks **Send to Parts**, the system:

- Sends an email with the service ticket information to one of the following addresses:

- **hvac-parts@makeitmowery.com**
- **plumbing-parts@makeitmowery.com**

- Logs a timestamp on the Service Call screen showing when the request was sent

### 🔄 Send to Parts

1. From the main menu, go to **Service Call**s.
2. In the right column (highlighted in blue), click a service card to open the record.
3. Scroll to **Step 4** (bottom right).
4. Click **Send to Parts**.
- The **Send to Parts** field updates with your name and timestamp.
    
- An email is sent to the correct department.
    
- A timestamp displays on the Service Call screen.

## 👥 View and Manage Staff Records

Manage employee records from the **Service Calls** module.

### View the Staff List

1. Click **Service Calls**.  
2. Select **Staff**.  
3. The **Staff List** shows all employees (read-only).
### Edit an Employee Record

1. In **Staff List**, click **Form View** (number to the left of the employee’s name).  
2. The **FileMaker Access** layout opens.  
3. Edit details. Changes save automatically.  

<p style="float: right; margin-left: 15px; width: 20%;">
  <img src="https://github.com/Fx-Professional-Services/MoweryDocs/blob/staging/Mowery/Mowery%20User%20Guide/Images/edit_an_employee_record.png" 
       alt="Edit an Employee Record" 
       style="width: 60%;" />
</p>
### Add an Employee

1. In **FileMaker Access**, click **New**.  
2. Enter the employee’s details. Changes save automatically.  
### Delete an Employee Record

- From **Staff List**, click **Delete Account** to remove the FileMaker account.  
- Click the red **(X)** to remove the employee record and history.  
- From **FileMaker Access**, click the **(X)** to delete the employee. Confirm when prompted.
