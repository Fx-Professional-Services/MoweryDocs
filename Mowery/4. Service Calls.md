# 🛠️ Service Calls

The **Service Calls** section is used to manage all service jobs in the system. These may include installation, repair, maintenance, or inspection work for HVAC, plumbing, or other services provided to a customer.

### 🔍 Purpose

A **Service Call** is a scheduled visit to a customer’s property for specific service work.

### 🛠️ Key Features

- Sort and manage scheduled jobs  
- Create or edit service calls  
- Print customer service history  
- Print service tickets  
- Quick view of active service jobs  
- Log service-related communications, including text messages to technicians  

These tools help you:

- Assign technicians using the Dispatch Board  
- Send technician details to customers via text message  
- Provide customers with a secure Technician Profile Page  
- Print a job-specific parts list  
- Manage technician and staff records  
- Understand system auto-logout behavior  
- Organize categories and subcategories for product assignments

## 🧑‍🔧 Assign Technicians Using the Dispatch Board

The **HVAC Schedule Board** allows dispatchers to assign technicians to unassigned service calls quickly and accurately.

### Assign a Technician to a Service Call
To assign a technician from the **Schedule Board**:

1. From the main menu, select **Service Calls**.
    
2. The **Schedule Board** opens. Unassigned HVAC calls appear in the **Unassigned** column.
    
3. Select a call in the **Unassigned** column to open the **Service Call Detail** screen.
    
- Alternatively, click **List View** (upper left).

On the HVAC board, assigned calls may not immediately appear under a technician’s column. If this happens, use the **Refresh** button to update the board and display the latest assignments.

This behavior is specific to the HVAC board.

## 📱 Send a Text Message with Technician Details

You can send customers a personalized text message with technician and appointment details, but only after certain conditions are met.

### 🔒 Requirements

- The service call **must be confirmed**.  
  - In **Step 2: Assign Technician**, the dispatcher must click the **blue arrow** and enter the time under **Confirm**.  
- A technician must be **assigned to the service call**.  
  - If no technician is assigned, clicking the **Send Text Message** icon will result in a pop-up:  
    **“No Technician selected. Please assign a technician before sending a message.”**
- The customer's **phone number** must be:  
  - Entered on the **Master Property List**  
  - Marked as **Preferred**  
  - Checked with **Accept Text Message**  
  - A valid **10-digit U.S. format**

### ✅ How to Send a Text Message

1. From the main menu, go to **Service Calls**.  
2. Open the scheduled and confirmed **service call**.  
3. In **Step 2: Assign Technician**:  
   - Verify a technician is assigned  
   - Confirm the call by entering the confirmation time via the blue arrow  
4. In the **Contact** section:  
   - Ensure **Text** is selected  
   - Phone number is valid and marked **Preferred**  
   - **Accept Text Message** is checked  
5. Click the **Send Text Message** (blue phone icon).  
6. Wait for the confirmation or error message.

### ✉️ Message Contents 

The customer receives a message with:

- **Technician** name (singular, not plural)  
- **Customer address**  
- Secure link to the **Technician Profile Page**  
- Technician introduction  
**Example message:**  
Hello [Customer Name], [Technician Name] from Mowery is on the way to [Customer Address].
Click here for more details: [Technician Profile Webpage Link] Hi

Hi, my name is [Technician Name]. I’ll be your technician today. [Technician Bio] If you have any questions, I’m here to help. Thank you for choosing Mowery!

### 🔤 Template Variables

| Variable                           | Description                              |
|-----------------------------------|------------------------------------------|
| `[Customer Name]`                 | Full name of the customer                |
| `[Technician Name]`               | Name of the assigned technician          |
| `[Customer Address]`              | Service location                         |
| `[Technician Bio]`                | Technician's short introduction          |
| `[Technician Profile Webpage Link]` | Secure link to the technician profile |

### ⚠️ Error Messages and Validations

| Condition                                 | Message                                                                |
| ----------------------------------------- | ---------------------------------------------------------------------- |
| No assigned technician                    | **No Technician selected. Please assign a technician before sending.** |
| No preferred number                       | A preferred phone number is required to send a message.                |
| Accept Text Message checked but no number | Cell number is required if Accept Text Message is checked.             |

## ✅ How a Customer Confirms the Service Call

When customers receive a service notification, it includes a link to the **Technician Profile Page**, customized for their service call. Clicking the link confirms the appointment, and the system logs the confirmation along with a timestamp. Each link is unique to a specific service call and expires once the call is closed.

### 🔍 What Customers See

**Service Information**  
- Service status (Scheduled, In Progress, Completed)  
- Appointment time  

**Technician Details**  
- Technician name  
- Technician photo

## 📄 Send a Parts List from a Service Call

As a dispatcher, you can email a parts request directly from the Service Call screen. This ensures the parts department is notified quickly and that a digital record of the request is saved.

  
### ✅ What Happens When You Click Send to Parts 

When a dispatcher clicks **Send to Parts**, the system:

- Sends an email with the service ticket information to one of the following addresses:

- **hvac-parts@makeitmowery.com**
- **plumbing-parts@makeitmowery.com**

- Logs a timestamp on the Service Call screen showing when the request was sent

This provides both a communication trail and a record of the action.


### 🔄 Use Send to Parts

1. From the main menu, go to **Service Call**.
2. In the rightmost column (highlighted in blue), click a service card to open the Customer Property or Service Call record.
3. Scroll to the **Step 4** section in the bottom-right corner.
4. Click **Send to Parts**.

- The **Send to Parts** field (bottom left) updates with your account name and the timestamp.
- An email with the service ticket information is sent to the appropriate parts department address.
- A timestamp appears on the Service Call screen showing when the request was sent.

## 👥 View and Manage Staff Records

Manage employee records from the **Service Calls** module.

### View the Staff List

1. Click **Service Calls**.  
2. Select **Staff**.  
3. The **Staff List** layout displays all employees (read-only).  
### Edit an Employee Record

1. In **Staff List**, click the **Form View** button (number to the left of the employee’s name).  
2. The **FileMaker Access** layout opens.  
3. Edit employee details. Changes save automatically.  

<p style="float: right; margin-left: 15px; width: 20%;">

       alt="Edit an Employee Record" 
       style="width: 60%;" />

### Add a New Employee

1. In **FileMaker Access**, click **New**.  
2. Enter the employee’s details. Changes save automatically.  
### Delete an Employee Record

- From **Staff List**, click **Delete Account** to remove the FileMaker account.  
- Click the red **(X)** to remove the employee record and history.  
- From **FileMaker Access**, click the **(X)** to delete the employee. Confirm when prompted.
