## Overview

The **Master Supply List** screen allows you to view, manage, and maintain all inventory supply items in the system. From this screen, you can:

- Add new inventory items
    
- Search and filter existing items
    
- Sort item records
    
- Review pricing and cost data
    
- Print audit worksheets
    

This guide explains each section of the screen and provides step-by-step instructions for common tasks.

---

## Screen Layout

The screen contains the following main areas:

1. **Top Navigation Bar**
    
2. **Action Toolbar**
    
3. **Inventory Grid**
    
4. **Record Navigation Controls**
    

---

## Action Toolbar

The toolbar directly under the title **Master Supply List** includes:

|Button|Description|
|---|---|
|**New Item**|Creates a new inventory record|
|**Find**|Searches for a specific item|
|**Find All**|Displays all items|
|**Sort**|Sorts the inventory list|
|**Print Audit Worksheet**|Prints an inventory audit report|
|**QuickFind Search**|Searches by keyword or SKU|

---

## Inventory Grid

The grid displays all supply items in a tabular format.

### Column Definitions

|Column|Description|
|---|---|
|**SKU #**|Unique system-generated item number|
|**Quantity**|Current stock quantity|
|**Unit Ms**|Unit of measure (EA, FT, PC, etc.)|
|**Max**|Maximum stock level|
|**Min**|Minimum stock level|
|**Modified On**|Date the item was last updated|
|**Description**|Item name or details|
|**Model #**|Manufacturer model number|
|**Mowery Part #**|Internal part reference|
|**Cost**|Unit cost|
|**Vendor**|Supplier name|
|**Sort #**|Sorting or categorization number|
|**Total Cost**|Total value of on-hand inventory|
|**Mowery Selling $**|Selling price|
|**2X Cost**|Double the cost (pricing reference)|
|**Red X icon**|Deletes the item|

# Common Tasks

---

## Add a New Item

1. Click **New Item**.
    
2. Enter the required information:
    
    - Description
        
    - Unit of measure
        
    - Cost
        
    - Vendor
        
    - Min and Max levels
        
3. Save the record.
    

Best practice:  
Set **Min** and **Max** levels when creating an item to support proper inventory control.

---

## Search for an Item

### Option 1: QuickFind

1. Enter a SKU number, description keyword, or model number in the **QuickFind Search** field.
    
2. Press **Enter**.
    

### Option 2: Find

1. Click **Find**.
    
2. Enter search criteria.
    
3. Confirm the search.
    

Use **Find All** to reset the grid and display all records.

---

## Sort Items

1. Click **Sort**.
    
2. Choose the field to sort by (for example, Vendor or SKU).
    
3. Apply the sort.
    

Sorting helps group related inventory items for review.

---

## Print an Audit Worksheet

1. Click **Print Audit Worksheet**.
    
2. Review print settings.
    
3. Print the report.
    

Use this report during physical inventory counts.

---

## Edit an Item

1. Locate the item in the grid.
    
2. Click into the field you want to change.
    
3. Update the value.
    
4. Save changes.
    

Tip: Always verify cost and selling price after editing.

---

## Delete an Item

1. Click the red **X** icon at the far right of the row.
    
2. Confirm deletion.
    

Warning: Deleting an item permanently removes it from the system. Confirm that the item is not used in active transactions before deleting.

---

# Inventory Management Best Practices

- Review **Min** and **Max** levels quarterly.
    
- Use consistent naming conventions in descriptions.
    
- Verify vendor and cost data when updating pricing.
    
- Use **Total Cost** to monitor inventory value.
    
- Run audit worksheets regularly.
    

---

# Troubleshooting

## Item Does Not Appear in Search

- Use **Find All** to reset filters.
    
- Check spelling in QuickFind.
    
- Search by SKU instead of description.
    

## Incorrect Inventory Quantity

- Verify recent transactions.
    
- Confirm physical count.
    
- Update the quantity if necessary.
    

---

# Summary

The Master Supply List is the central location for managing supply inventory. Use the toolbar for searching and reporting, and maintain accurate Min/Max levels to support efficient stock control.
