## ➕ Create a Catalog (Price Book)

To create a new catalog in Mowery:

1. Navigate to **System Defaults → Catalog**.
2. Click the **New Catalog** button.
3. A card window will appear. Enter the following details:
   - **Effective Date**
   - **Title**
   - **Revision**
   - Any other required fields
4. Click the **Add Catalog** button to create the catalog.
5. A dialog will appear stating that catalog creation is in progress. Click **OK**.
6. Once processing is complete, another dialog will confirm whether catalog creation succeeded or failed.
7. Click the catalog name to open and view the catalog items.
8. Ensure that the number of catalog items matches the number of active products.

> 📘 Catalogs in Mowery are also known as **price books**. Each catalog reflects a list of product prices effective from the specified date.
