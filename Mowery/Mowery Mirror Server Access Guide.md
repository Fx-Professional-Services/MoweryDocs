# 🖥️ Mowery Mirror Server Access Guide

This guide explains how to access the Mowery Mirror Server via LogMeIn and log in to the Mowery Mirror Admin Console.

---

## 🔐 Accessing the Mowery Mirror Server

1. Send an email to [ethan.strange@scgtechnologies.com](mailto:ethan.strange@scgtechnologies.com) to request access to the Mowery server.      
2. After receiving an invitation, click the link in the email.
3. Follow the prompts to create a **LogMeIn** account.
4. Set up **Two-Factor Authentication (2FA)** when prompted.
5. Once your account is created, you'll be redirected to the LogMeIn login page.
6. Sign in using your account email, password, and 2FA code.
7. After successful login, you’ll be taken to the **LogMeIn Computers** page.
8. In the list of computers, click **"Mowery - FMMirror"**.
9. A remote Windows login screen will appear.
10. Enter the username and password provided in **Keeper**.

---

## 🛠️ Accessing the Mowery Mirror Admin Console

1. Open your web browser and go to: [https://fm2.moweryservice.net/admin-console](https://fm2.moweryservice.net/admin-console)
2. Enter the username and password stored in **Keeper**.
3. Click **Sign In**.

---

## 📌 Notes

- Make sure you have access to **Keeper** to retrieve credentials.
- If you encounter any issues, contact Ethan Strange for support.
