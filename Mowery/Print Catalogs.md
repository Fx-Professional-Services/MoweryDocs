## Overview

This guide explains how to create and print catalogs (also known as price books) in Residential or Commercial formats. You can preview the catalog before printing to ensure it is accurate and up to date.

---

## Glossary

| Term | Definition |
|------|------------|
| **Catalog** | A list of products and their pricing details used for creating price books. |
| **Price Book** | A formatted, printable version of a catalog, typically used by sales or operations. |
| **Effective Date** | The date when the catalog's pricing and content become valid. |
| **Revision** | A version identifier used to track updates to the catalog. |
| **System Defaults** | A section in the software where global or system-level configurations are set. |

---

## ✅ Prerequisites

- A catalog must already exist in the system.
- You must have permission to view and print catalogs.

---

## ➕ Create a Catalog

1. Navigate to **System Defaults → Catalog**.
2. Click **New Catalog**.
3. Fill out the catalog form:
   - **Effective Date**
   - **Title**
   - **Revision**
   - Any other required fields
4. Click **Add Catalog**.
5. When prompted, click **OK** to confirm catalog creation.
6. Wait for a confirmation message indicating success or failure.
7. Click the catalog name to open it.
8. Verify that the number of items matches the number of active products.

---

## 🖨️ Print a Price Book

1. Go to **System Defaults → Catalog**.
2. Select a catalog:
   - For a new catalog, follow the steps above.
   - For an existing catalog:
     - Locate it in the list.
     - Click the **leftmost button** next to the **Code** column to open it.

3. Click **View Catalog**.

4. Click **Print**.
5. Select a format:
   - **Residential**
   - **Commercial**

<img src="https://github.com/Fx-Professional-Services/MoweryDocs/blob/1e36d159fcf3d5d763fe6d3b338c804a0fb539c6/Mowery/3_3_print_catalog.png" width="350" height="230">

4. A print preview window opens.
5. Review the preview:
   - Product names
   - Pricing details
   - Page layout and formatting
   - **Revision number**, **page number**, and **effective date** on each page
8. If everything looks correct, use your system’s print dialog to print or export to PDF.

## ✅ Expected Result

- A formatted print preview of the price book (Residential or Commercial) is displayed.
- You can verify and print the content, or export it as a PDF for sharing.
