
# 📘 Mowery FileMaker – Staff Management

This article explains how to use the **Staff List** and **FileMaker Access** layouts in the Mowery FileMaker system to view, add, edit, and delete employee records.

---
## 📋 Overview

- The **Staff List** layout provides a read-only view of all employees.

- The **FileMaker Access** layout allows users to add, edit, and delete employee records.

- Clicking an employee’s name or form view button redirects to the FileMaker Access layout for editing.

---

## 👁️ View Staff List

To view the list of employees:

1. From the main menu, click **Service Calls**.

2. In the **Service Calls** layout, click **Staff**.

3. The **Staff List** layout appears, showing a list of all employees.

### Behavior

- This layout is **read-only**.

- Clicking an employee’s **name** opens the employee record in the **FileMaker Access** layout.

- Clicking the **Form View** button (left edge next to the row number) also opens the record in FileMaker Access.

---
## ✏️ Edit an Employee

To edit an existing employee record:

1. From the main menu, click **Service Calls**.

2. Click **Staff** to open the Staff List.

3. Click the employee’s **name** or the **Form View** button.

4. The system redirects you to the **FileMaker Access** layout.

5. Make the necessary updates.

6. Changes are automatically saved.

---
## ➕ Add an Employee

To add a new employee:

1. From the **FileMaker Access** layout, click the **New** button in the top-right corner.

2. Enter the employee’s information.

3. The created employee record automatically save 

> ℹ️ This process ensures that new employee records follow the latest workflow and validation rules.

---
## 🗑️ Delete an Employee

To delete an employee record:

1. From the **FileMaker Access** layout, select the employee.

2. Click the **(x)** button to the far right of the employee's name row.

3. A confirmation dialog will appear. Choose **Delete** to proceed or **Cancel** to abort.
